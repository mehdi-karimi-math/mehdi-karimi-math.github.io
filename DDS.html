DDS (Domain-Driven Solver) is a Matlab-based code for convex optimization problems given in the Domain-Driven formulation . In principle, every convex optimization problem can be treated in the Domain-Driven setup. The current version of DDS solves a large class of problems including LP, SOCP, SDP, QCQP, nuclear-norm minimization, geometric programming, entropy programming, and more is being added.
